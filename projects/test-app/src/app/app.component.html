Toasty

Lightweight and flexible toast notifications for Angular

GitHub â†—

Install

npm install ng-toasty

Add component

component.html
<toasty [position]="containerPosition" [duration]="containerDuration" [capacity]="containerCapacity"></toasty>
Position, capacity and duration are optionally
component.ts 
import { ToastType, ToastyComponent, ToastyPromise, ToastyService } from 'toasty'; 
@Component({
  selector: 'app-root',
  standalone: true,
  imports: [ToastyComponent],
  templateUrl: './app.component.html',
  styleUrl: './app.component.css'
})
Component parameters
Name Type Default Description
position string "bottom-right" Controls the toast position. Options: top-left, top, top-right, bottom-left, bottom, bottom-right.
duration number 5 Time in seconds before a toast disappears automatically.
capacity number 5 Maximum number of concurrent toasts allowed.

Service

constructor(private _toastyService: ToastyService) { }
this._toastyService.showToast('Title', 'Message body');
