HTML Message

You can use HTML content in your toast message if you indicate enableHtml: true as option.

Custom CSS Toast

It is possible to create custom css rules indicating type: Custom and customStyle parameters. Note that you can define key:value pairs just as you would in a CSS document inside a variable and assign them to customStyle: style

Long Message Toast

Toast is adapted vertically in base of the content length.

Loading Icon Toast

Aditionally you can add a loding icon, just setting loading: true in the configuration parameter.

Close on swipe

You can close any toast by swiping it horizontally. This is useful for mobile devices.

Beep sound

You can enable a beep sound when a toast is displayed. This is useful for notifications that require attention. By default beeps are false

Progress Bar

You can add a progress bar to your toast. It will animate while the toast is visible.

Sticky Toast

You can create a sticky toast that will not disappear automatically. The user must close it manually.

Component Toast

You can create a toast using a custom Angular component. This allows you to have full control over the content and behavior of the toast. Just create your component and use the ToastType.Component type, indicating the component and optionally the input parameters. Just use showToastComponent method from service.
Here is an example using a simple custom component that displays a clock.

Scheduled Toast

You can schedule a toast to appear at a specific time in the future, and optionally repeat it at regular intervals. To do this, use the showToastSchedule method from the service, providing a ToastSchedule object with the desired settings. Component parameters
Name Type Default Description
position string "bottom-right" Controls the toast position. Options: top-left, top, top-right, bottom-left, bottom, bottom-right.
duration number 5000 Time in mili-seconds before a toast disappears automatically.
capacity number 10 Maximum number of concurrent toasts allowed.
grouping boolean false Allow same toasts group together
